    Getting a list of metrics

    Getting a list of metrics > Getting a list of metrics
    Method get.metrics
    Description Getting a list of metrics
    Available to Agent, Client
    Necessary components for working with the method Data API for configuration
    Method weight 1 point

    Request parameters

    Title Type Required Description
    access_token string yes Authentication session key
    user_id number no Unique user identifier of the agent client on behalf of which the request is made
    Is required for the agent
    To obtain a list of client users, you must use the method "get.customer_users"
    report_type enum yes User report type. To obtain a list of user reports, you must use the method "get.reports_list"

    Response parameters

    Name Type Valid values Default response Description
    id enum yes Unique metric mnemonic
    name string yes Metric name
    data_type enum yes Data format
    List of aggregates available for the metric
    aggregates array yes List of aggregates available for the selected metric
    id enum Unique unit mnemonic
    name string Unit name
    has_percentage boolean Is it possible to calculate the share
    is_default boolean Default unit
    List of multichannel analysis methods available for metrics
    multichannel_models object yes List of multichannel analysis methods available for metrics.
    Methods of multichannel analysis
    models array yes Methods of multichannel analysis.
    id enum Unique mnemonic of multi-channel analysis
    name string Name of multichannel analysis
    description string Description of multichannel analysis
    is_default boolean Multichannel analysis by default
    group_id string Multi-channel analysis group
    sort number End-to-end identifier used for sorting in the tree.
    Multichannel analysis groups
    models_groups array yes List of multichannel analysis groups
    id string Mnemonics of the multi-channel analysis group
    name string Multichannel analysis group name
    description string Description of the multichannel analysis group.
    parent_group_id number Identifier of the group in whose attachment this multichannel analysis group is located
    sort number End-to-end identifier used for sorting.

    List of possible units

    Mnemonics Name Is share calculation used Description
    count Quantity Yes
    distinctcount Number of unique ones Yes
    sum Amount Yes
    avg Average
    median Median
    min Minimum
    max Maximum
