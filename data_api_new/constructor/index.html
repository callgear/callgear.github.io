    Getting account information

    Method Description
    "getobj.account" Getting account information

    User reports

    Method Description
    "get.reports_list" Retrieving all user reports
    "create.report" Create a user report
    "update.report" Editing a User Report
    "delete.report" Delete a user report

    Report status

    Method Description
    "getobj.report_state" Getting the current report status
    "set.report_state" Change report status
    "create.shared_report_settings" Create current shared report settings

    Receiving data

    Method Description
    "getobj.report_table" Getting table data in a user report
    "getobj.report_total" Getting the resulting data in a user report
    "getobj.report_transfer" Method for moving from an aggregated report to a flat report
    "get.communications" Getting a list of all requests
    "get.deals" Getting a list of all transactions
    "get.visits" Getting a list of all visits in the last 30 minutes
    "get.chats" Getting a list of chats

    Uploading reports

    Method Description
    "create.report_generator_task" Creating a task to upload a report as a file
    "get.report_generator_tasks" Obtaining a list of tasks for uploading reports with a file

    Global calendar

    Method Description
    "get.report_global_calendars" Retrieving a user's global calendars
    "update.report_global_calendar" Editing the global calendar

    Report groups and types

    Method Description
    "get.report_groups" Retrieving report groups
    "get.report_types" Getting report types

    Filters

    Method Description
    "get.report_filters" Getting a list of possible filtering conditions for a report
    "get.report_rapid_filters" Getting a list of possible quick filter conditions for a report
