    Getting a list of possible filtering conditions for a source

    Sources > Getting a list of possible filtering conditions for a source
    Method get.source_filters
    Description Getting a list of possible filtering conditions for a source
    Available to Agent, Client
    Necessary components for working with the method Data API for configuration
    Method weight 1 point

    Request parameters

    Title Type Required Description
    access_token string yes Authentication session key
    user_id number no Unique user identifier of the agent client on behalf of which the request is made
    Is required for the agent
    To obtain a list of client users, you must use the method "get.customer_users"

    Response parameters

    Name Type Valid values Default response Description
    id enum yes Unique filtering condition mnemonic
    name string yes Filtering condition name
    description string yes Description of filtering condition
    format enum yes
    Filtering operators
    operators array yes Filtering operators available for the selected condition. A list of all operators can be obtained in the section "Filtering operators"
    id enum Filtering operator mnemonic
    name string Filtering operator name
    is_default boolean Default filter operator
    List of possible values
    values array yes Returns either a list of possible values for the selected condition, or a list of Top 10 values from statistics for the last 7 days

    JSON request structure```json

    { "jsonrpc":"2.0", "id":"number", "method":"get.source_filters", "params":{ "access_token":"string", "user_id":"number" } }
