    comment string no Comment on the call. You can specify the name of the service or form that generated the call.

    Requirements for wav files for speech analytics

  11. Record a call in stereo mode with two audio channels: employee speech in one channel, client speech in another channel.
  12. 16 bit PCM
  13. Sampling rates: 8000 Hz
  14. Frequency range no worse than 330-3400 Hz;
  15. Signal-to-noise ratio in the frequency range 330-3400 Hz: no less than 15 dB;
  16. Minimum duration of speech signal 16 seconds;
  17. File size no more than 50Mb

    18. Requirements for the service that will provide call recordings when downloading them from the provided URL

    • The response to a request to download wav files must contain the Content-Type: audio/wav header
    • In response to a request to download mp3 files there must be a Content-Type header: audio/mpeg

    ### JSON response structure
<blockquote>The response comes as soon as the system has saved the calls for further processing. When the call is processed, a web-hook will be sent with the processing status.</blockquote>```json
{
  "jsonrpc": "2.0",
  "id": "number",
  "result": {
  "data": {
      "success": "true"
    }
  }

            "tag_name": "Тест"

            "employee_channel": "left",

    List of returned errors

    Parameter Meaning
    max_file_size_download_error The maximum file size has been exceeded. Currently - 50 MB.
    decode_error Could not open the conversation file, the first bytes are missing in the mp3 - ID3 and wav - RIFF files, or an error occurred during the decoding process.
    wav_min_duration_processing_error The downloaded wav file is shorter than 16 seconds.
    wav_channels_processing_error The downloaded wav file does not contain 2 channels.
    wav_processing_error Error when processing wav file when encoding the file to the required parameters.
    connection_download_error Error connecting to the server. Network problem.
    http_status_download_error The server from which we downloaded the file did not respond 200.
    http_content_type_download_error Content-Type is out of specification.
    timeout_download_error It took a very long time to download a file or a very long time to connect to the server. Maximum time - 30 seconds

    See the section "List of errors for methods with the create verb"
