    Editing groups of phone numbers to replace with call tracking numbers

    Sites > Editing groups of phone numbers to replace them with call tracking numbers
    Method update.replacement_phones_group
    Description Editing groups of phone numbers to replace them with call tracking numbers
    Available to Agent, Client

    Request parameters

    Name Type Required Valid values Description
    access_token string yes Authentication session key
    user_id number no Unique user identifier of the agent client on behalf of which the request is made
    Is required for the agent
    To obtain a list of client users, you must use the method "get.customer_users"
    id number yes Unique identifier of the group of elements to be replaced
    name string Group name. The name tells you which number the group is responsible for. For example, you can specify the name of the department: sales department or technical department. support
    site_ids array Unique identifiers of sites that own groups of site elements to be replaced with call tracking numbers
    Array of site page element selectors, automatically selected call tracking numbers
    phone_selectors array Within the context of one site (and group) there should not be selectors with the same element_attribute and element_attribute_value pairs Array of site page element selectors, automatically selected call tracking numbers
    element_attribute enum yes
    • id
    • class
    • name
    • selector
    • number
    		 Attribute of the replaced element on the site page
    • id - Identifier;
    • class - Class;
    • name - Title;
    • selector - Selector;
    • number - Number on the site.
    element_attribute_value string yes Name, value of the replaced element on the site page
    phone_number_template string no Number format. Choose how the phone number will be displayed in the {phone} tag. The digits of the number are indicated by the # symbol; there can be from 5 to 7. If your number has a different number of digits, they will remain at the beginning of the mask. For example, if the mask ###-##-## is selected, the number with extra characters 1234567 will be displayed as 123-45-67. A number with missing digits 12345 will be displayed as 1-23-45.
    phone_number_mask string no Phone mask. The rule for displaying call tracking numbers on the website. Choose from existing templates or customize your own by using tags or adding any text. Tag values: {country} - country code, {city} - city code, {phone} - phone number. Example: +{country} ({city}) {phone}
