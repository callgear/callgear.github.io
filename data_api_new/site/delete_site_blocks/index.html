    Deleting a block of numbers for a site

    Sites > Removing a block of numbers for a site
    Method delete.site_blocks
    Description Removing a block of numbers for a site
    Available to Agent, Client

    Request parameters

    Title Type Required Description
    access_token string yes Authentication session key
    id number yes Unique number block identifier
    user_id number no Unique user identifier of the agent client on behalf of which the request is made
    Is required for the agent
    To obtain a list of client users, you must use the method "get.customer_users"

    JSON request structure```json

    { "jsonrpc": "2.0", "id": "number", "method": "delete.site_blocks", "params": { "access_token": "string", "user_id": "number", "id": "number" } }

    ### JSON response structure```JSON
{
  "jsonrpc": "2.0",
  "id": "number",
  "result": {
      "data": {},
      "metadata": "object"
  }
}
