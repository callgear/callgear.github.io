    The maximum period for uploading a report is 90 days

    List of available methods

    General reports

    Method Description
    "get.communications_report" Getting a list of all requests
    "get.campaign_daily_stat" Obtaining statistics of the Republic of Kazakhstan by day
    "get.employee_stat" Aggregated employee statistics
    "get.communications" Communications report, available 366 days
    "getobj.report_table" Getting table data in a user report, available for 366 days
    "getobj.report_total" Obtaining the resulting data in the user report, available for 366 days
    "get.visits" "Visits" report. Getting a list of all visits for 31 days

    Calls

    Method Description
    "get.calls_report" Receiving a report on call sessions
    "get.call_legs_report" Getting a list of CDRs for a call session
    "get.financial_call_legs_report" Obtaining a detailed report on the history of charges for all arms of calling sessions

    Chats

    Method Description
    "get.chats_report" Getting information about chats
    "get.chat_messages_report" Receive all chat messages
    "get.chat_channels_report" Retrieving all chat channels

    Applications

    Method Description
    "get.offline_messages_report" Obtaining information on offline applications

    Goals| Method | Description |

    | ------------------------------------------------------------------------------ | ------------------------------------------------------------------------ | | "get.goals_report" | Getting a list of achieved goals |

    Sessions

    Method Description
    "get.visitor_sessions_report" Getting information about a visitor's session

    Coltracking

    Method Description
    "get.ct_ac_summary_report" Call tracking numbers for advertising campaigns
    "get.ct_summary_report" Call tracking numbers by site

    Helper methods

    Method Description
    "getobj.columns_tree" Getting column nesting
    "get.reports_list" Retrieving all user reports
    "get.report_filters" Getting a list of possible filtering conditions for a report
    "getobj.dimensions_tree" Getting dimension nesting
    "get.report_types" Getting report types
