    Getting CRM source settings

    Getting CRM source settings > Getting CRM source settings
    Method getobj.crm_source
    Description Getting CRM source settings
    Available to Agent, Client
    Necessary components for working with the method Data API for configuration
    Method weight 1 point

    Request parameters

    Title Type Required Description
    access_token string yes Authentication session key
    user_id number no Unique user identifier of the agent client on behalf of which the request is made
    Is required for the agent
    To obtain a list of client users, you must use the method "get.customer_users"
    id number yes Unique identifier of the detail source. To get a list of sources, you must use the "get.sources"

    Response parameters

    method
    Name Type Valid values Default response Description
    id number yes CRM source ID
    name string yes CRM source name
    description string yes Description of CRM source
    source_id number yes User Source ID. To get a list of Sources you must use the "get.sources"
    source_name string yes Name of the Custom Source
    creation_date iso8601 yes YYYY-MM-DD hh:mm:ss CRM source creation date
    state enum
    • active CRM Source active
    • off CRM Source disabled
    		Source CRM status
    priority number yes yes End-to-end identifier used for sorting.
    Filter for conditions used in a custom source
    filter object yes Filtering conditions used in the custom source. To get a list of available filtering conditions, see section ["Getting a list of possible filtering conditions for a source"](../get_crm_source_filters)
    filters array An expression that can contain both simple filters and a filter tree
    field string Field of the entity to which filtering will be applied (the list is predefined for the method)
    operator enum Filtering operator. A list of all operators can be obtained in the section ["Filtering Operators"](../index.md#_filterOperators)
    value string Value for the filtering operator. Optional field; if it is missing, it is considered empty.
    entity string
    • dealCRM source based on Transaction parameters
    • leadCRM source based on Lead parameters
    		 CRM source mnemonic determined by the filtering parameters used
