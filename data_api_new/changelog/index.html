    Changelog

    2025-04-11 Methods for call processing rules

    • Call processing rules have been added to the get.virtual_numbers method

    Method Description
    "getobj.call_processing_rules" Obtaining call processing rules
    "set.call_processing_rule_is_active" Setting the call processing rule activity

    2023-07-31 Adding ext_id

    • Added the ability to download Email messages

    Method Description
    "upload.emails" Loading email

    2023-05-31 Adding ext_id

    • Added the ability to work with communications by ext_id

    Method Description
    "set.tag_communications" Adding a tag to a request
    "unset.tag_communications" Removing a tag from circulation
    "upload.chats" Loading chats
    "upload.offline_messages" Uploading offline applications
    "get.calls_report" Receiving a report on call sessions
    "get.chats_report" Getting information about chats
    "get.offline_messages_report" Obtaining information on offline applications

    2021-06-28 Loading expenses for advertising campaigns

    • Added a new method for loading expenses for advertising campaigns

    | Method | Description || ------------------------------------------------------------------------------ | ------------------------------------------------------------------------ | | "upload.campaign_costs" | Loading expenses for advertising campaigns |

    2021-03-22 New Call Tracking reports

    • Added new methods for obtaining the “Call Tracking Numbers” report broken down by advertising campaigns and sites.

    Method Description
    "get.ct_ac_summary_report" Call tracking numbers for advertising campaigns
    "get.ct_summary_report" Call tracking numbers by site

    2021-03-04 Added the name field to the methods for creating, editing, and getting a list of sites

    Method Description
    "create.sites" Website creation
    "get.sites" Getting a list of sites
    "update.sites" Website editing

    2021-02-08 Blacklist

    • Changes have been made to the DataAPI documentation structure
    • Added work with the Blacklist

    Method Description
    "get.blacklist_phone_numbers" Getting blacklisted phone numbers
    "delete.phone_number_from_blacklist" Removing a phone number from the blacklist
    "create.phone_number_to_blacklist" Adding a phone number to the blacklist

    2019-11-14

    Added to "Loading communications"

    • Loading/deleting offline applications upload.offline_messages
    • Loading/deleting chats upload.chats

    [2.0.9] - 2019-10-03

    Added

    • Getting an account get.account

    [2.0.8] - 2019-01-10

    Added

    • Website creation create.sites
    • Deleting the site delete.sites
    • Website editing update.sites

    [2.0.7] - 2018-11-29

    Added

    • Obtaining statistics of the Republic of Kazakhstan by day. Method get.campaign_daily_stat
    • Receiving a list of virtual numbers available for connection get.available_virtual_numbers"- Connecting a virtual number enable.virtual_numbers
    • Disable virtual number disable.virtual_numbers
    • Added parameter include_ongoing_calls "Enable current calls" for the get.calls_report method

    [2.0.6] - 2018-08-10

    Added

    • Added parameters for managing employee status and communication permissions to the create.employees, get.employees, update.employees methods: status, allowed_in_call_types, allowed_out_call_types;
    • The calls_available parameter will be removed from the create.employees, get.employees, update.employees methods after 2 months. This parameter is not recommended for use.

    [2.0.5] - 2018-07-12

    Added

    • Added the lost call processing time parameter "lost_call_processing_duration" to the get.calls_report method;
    • Added a new method for obtaining a list of scripts get.scenarios;
    • Added a new method for receiving media files - system, user get.media_files.

    [2.0.4] - 2018-02-15

    Added

    • Added the ability to get a list of sites. Method get.sites;
    • Added the ability to obtain a list of site number blocks. Method get.site_blocks;
    • In the methods for creating an employee and a group of employees, the maximum length of the internal number has been expanded to 5 characters;
    • Added the ability to manage SIP lines. Methods get.sip_lines, create.sip_lines, update.sip_lines, delete.sip_lines, update.sip_line_password.

    [2.0.3] - 2018-02-07

    Added

    • Added the ability to obtain a list of connected virtual numbers. Method get.virtual_numbers;
    • Information about integrated goals has been added to the get.goals_report method. Parameters type, source, ext_id;
    • The region of the subscriber number has been added to the get.calls_report method. Parameter cpn_region_id, cpn_region_name;
    • In the get.call_legs_report method we added information about the group within which the call was made, as well as information about the operation within which the call was created. Parameters group_id, group_name, action_id, action_name.

    [2.0.2] - 2018-01-29

    Added

    • Methods of managing advertising campaigns - creating, deleting, receiving, editing;
    • Information about integrated advertising campaigns has been added to the “Manage reports -> get.visitor_sessions_report” method. Parameter integrated_campaign_data;
    • In the get.calls_report method, the allowed value of the finish_reason parameter has been expanded.
